강아지 공장이란?

상업적 목적으로 강아지를 사육하는 농장이다.

하지만 위생상태나 동물 복지 기준을

상당히 위반한 생산 시설을 의미 한다.

문제점

좁고 구멍이 뚫린 금속 케이지를 여러 개 늘어놓고 사육을 하며 더러운 생활환경속에서
모견들은 평생 생활한다.

또한 강제 교배를 하다가 모견이 건강상의 문제로 교배가 불가능해지면 업자가 직접 수컷 강아지의 정액을 강제로 채취한 후 대형 주사기로 암컷에게 주입하는 장면도 방송에 나오기도 했다.
이처럼 비윤리적인 행위가 나타나고 있다.

자세히 보기

실태

대한민국 2018년 기준 농림축산식품부에
등록된 개 번식장은 1186곳이다.
동물보호시민단체 '카라'는 우리나라에 약 3000~4000곳의 강아지 공장이 운영되고 있다고 주장하고 있다.
즉 애견시장은 지속적으로 성장 중이며 농림축산식품부에 등록이 안된 업 채가 2000곳이 넘는다.
국내에서 길러지는 애완견의 대부분은 강아지 공장에서 생산되는것으로 추정된다.

동물보호법 위반시
최대 3년 이하의 징역 또는 3000만원 이하의 벌금

유통과정
( 강아지 공장 )
- 비닐하우스 혹은 켄넬이라는 곳에서 모견들은 평생을 임신과 출산을 반복하며 산다.
( 경매장 )
- 2달도 안된 어린 강아지가 경매장으로 가서 돈으로 가치를 매겨진다.
( 펫샵 )
- 경매장에서 구입한 어린 강아지를 유리 칸막이에 넣어 강아지를 판매한다.
수요가 없으면 공급도 없습니다.
우리가 할 수 있는 일
1. 책임질 자신이 없으면 키우지 말기.
2. 만약 키운다면 분양보다 입양하기.
3. 키우게 된다면 강아지에 대하여 공부하기
4. 책임지고 데려온 가족을 유기견으로 절대 만들지 말기.
강아지는 물건이 아닙니다. 가족입니다. 가족처럼 생각해 주세요.
